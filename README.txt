Project Title: Campus Lost & Found Assistant

Main Python File:
campus_lost_found_assistant.py

How to Run:
python3 campus_lost_found_assistant.py

Description:
This is a terminal-based Python program that helps students report, search, and manage lost and found items on campus.

No external libraries are required.
The program only uses Python built-in libraries: json, os, and datetime.

The program can:
1. Report a lost item
2. Report a found item
3. View lost items
4. View found items
5. Search possible matches
6. Mark an item as returned
7. View returned items
8. Save and exit

The program automatically creates a lost_found_data.json file to save item records when it runs.
